TC8566AF driver
================

This license covers the TC8566AF FDC driver included in
nextor-tc8566af.rom.  That ROM combines the Nextor kernel (covered by
LICENSE-Nextor.md) with this driver.

The driver source code and the full integration project are available
at:

  https://github.com/Hesoten/cbios-nextor_FDD-and-turboR
