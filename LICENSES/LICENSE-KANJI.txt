KANJI.rom - "Kanji ROM image file for msx emulaters"
=====================================================

The bundled KANJI.rom is the freely redistributable Kanji ROM image
created by A to C:

  http://www.yo.rim.or.jp/~anaka/AtoC/labo/labo32.htm

It provides the JIS level 1 + level 2 Kanji font (256 KB, I/O ports
D8h-DBh) used by Japanese MSX software, without containing any data
copied from a real machine's Kanji ROM.
